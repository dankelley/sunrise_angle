The maps show Halifax, Nova Scotia, with gold-coloured lines indicating the direction of the sun's rays at sunrise and sunset, at two different map scales. These may be useful in planning commuting routes, to avoid driving into the sun.

The maps are updated at 2AM daily, with a script written in the R programming language.

The source code for this project is at [https://github.com/dankelley/sunrise_angle](https://github.com/dankelley/sunrise_angle). If you have problems altering the code for your own region, or notice that problems have cropped up on the present website, please open by visiting [https://github.com/dankelley/sunrise_angle/issues](https://github.com/dankelley/sunrise_angle/issues).
